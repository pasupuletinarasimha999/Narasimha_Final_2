Installation of EKS and ArgoCD
===============================
1. Download kubectl.exe and add it in env variable
2. Install aws cli
3. Execute this command in CMD : aws configure and add Access/Secret Key
4. Create a USER in AWS with Admin Privileges, get access and secret key
5. Configure in above command
6. Now move to folder which we want to run the terraform. give terraform init command to initialize terraform on that folder
7. Give terraform apply to create resources
